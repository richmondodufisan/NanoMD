Program behaviour: overwrite

Syntax

-overw

-ow

Description

This will make Atomsk automatically overwrite the output files if they exist. By default (i.e. when -ow is not used), the user is prompted if the file shall be overwritten or another file name shall be used.

If both -ow and -ignore are used, then the existing files are not overwritten but are ignored.

To make this change permanent, you may set up a configuration file.

Default

By default files that already exist are not overwritten; instead the user is prompted whether to overwrite them or choose another name.

Examples

  • atomsk initial.gin cfg -ow

    This will read initial.gin and convert it to CFG; if initial.cfg already exists, it will be overwritten.
