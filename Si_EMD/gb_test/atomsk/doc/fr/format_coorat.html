Format : COORAT

Nom : fichier de positions atomiques pour le code Mixed-Basis Pseudopotential (MBPP) code

Extension : aucune (le fichier doit être nommé COORAT pour une utilisation avec MBPP)

Specification : N/A

Programmes de visualisation : aucun

Restrictions

Les vecteurs de boîte ne sont pas spécifiés dans un fichier COORAT, mais dans le fichier d'input (INP) de MBPP. En lisant un fichier COORAT, Atomsk cherchera si un tel fichier INP existe dans le dossier courant, et le cas échéant s'en servira pour lire les vecteurs de boîte. Chaque composante de ces vecteurs peut contenir un caractère spécial (o, e, h...) et/ou une étoile (*) indiquant la multiplication avec un nombre (par ex. "0.5*h"); le signe slash (/) n'est pas reconnu par Atomsk. Si aucun fichier INP n'est trouvé alors Atomsk définit automatiquement une boîte (ceci est identique à l'option -rebox) pour assurer la compatibilité avec d'autres formats.

Lorsqu'il écrit un fichier COORAT, Atomsk tente d'écrire les vecteurs de boîte dans un fichier INP dans le dossier courant. Si un fichier INP existe déjà alors Atomsk demandera s'il faut l'écraser ou utiliser un autre nom de fichier. Notez que le fichier INP créé sera incomplet et inapproprié pour une simulation.
