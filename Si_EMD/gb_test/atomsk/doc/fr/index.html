  6. Options
    • add-atom - Insérer un ou plusieurs nouveaux atomes
    • add-shells - Créer des coquilles pour certains atomes
    • alignx - Aligner le premier vecteur de boîte avec l'axe X
    • bind-shells - Re-associer les coquilles avec leur cœur
    • cell - Modifier les vecteurs de boîte
    • center - Translater le système au centre de la boîte
    • crack - Insérer une fissure dans le système
    • cut - Supprimer une partie du système
    • deform - Appliquer une déformation uniaxiale ou un cisaillement
    • dislocation - Insérer une dislocation dans le système
    • disturb - Perturber les positions des atomes aléatoirement
    • duplicate - Dupliquer le système, construire des super-cellules
    • fix - Fixer les coordonnées de certains atomes
    • fractional - Convertir en coordonnées réduites
    • mirror - Appliquer une transformation miroir
    • options - Appliquer les options écrites dans un fichier
    • orient - Changer l'orientation crystallographique
    • orthogonal-cell - Trouver une boîte orthogonale équivalente
    • properties - Définir des propriétés du système
    • rebox - Tente de déterminer les vecteur de boîte
    • reduce-cell - Essaye de réduire la taille du système
    • remove-atom - Supprimer un ou plusieurs atomes
    • remove-doubles - Supprimer les atomes en double
    • remove-property - Supprimer une ou plusieurs propriétés auxiliaires
    • remove-shells - Supprimer les coquilles sur certains ions
    • roll - Plier ou enrouler le système autour d'un axe
    • rotate - Tourner le système autour d'un axe
    • round-off - Arrondir les coordonnées ou une propriété auxiliaire
    • select - Sélectionner les atomes à l'intérieur ou à l'extérieur d'une région
    • separate - Sépare les atomes qui sont trop proches
    • shift - Translater une partie du système
    • sort - Changer l'ordre dans lequel les atomes sont rangés
    • spacegroup - Appliquer les symétries d'un groupe d'espace
    • stress - Appliquer une contrainte au système
    • substitute - Remplacer des atomes par d'autres atomes
    • swap - Échanger deux atomes, deux axes, ou deux propriétés
    • torsion - Appliquer une torsion le long d'un axe
    • unit - Changer de système d'unités
    • unskew - Réduire le cisaillement de la boîte
    • velocity - Générer une distribution de vitesses de Maxwell-Boltzmann
    • wrap - Remballer les atomes dans la boîte
  7. Modes (cliquez pour en voir davantage)
    • Average - Moyenner les positions atomiques de plusieurs configurations
    • Copy-Properties - Copier les propriétés d'un systèem à un autre
    • Create - Créer un système à partir de rien
    • ddplot - Créer un fichier pour visualisation avec le programme ddplot
    • Gather - Rassembler plusieurs fichiers vers un seul contenant toutes les configurations
    • Liste - Convertir tous les fichiers d'une liste
    • Help - Afficher l'aide
    • Interactive - Manipuler des systèmes interactivement ou avec des scripts
    • Interpolate - Interpoler les positions des atomes entre deux configurations
    • Match-ID - Trouver des atomes équivalents entre deux systèmes
    • Merge - Joindre deux ou plusieurs systèmes en un seul
    • Normal - Convertir un fichier vers un ou plusieurs formats
    • Polycrystal - Créer un polycristal avec la méthode de Voronoi
    • Unfold - Décomposer un fichier contenant plusieurs systèmes vers plusieurs fichiers
    • Unwrap - Déballer les coordonnées atomiques
  8. Comportement du programme
    • ignore - Toujours ignorer les fichiers existants
    • language - Définir la langue des messages
    • logfile - Définir le nom du fichier journal
    • nthreads - Définir le nombre de processus parallèles OpenMP
    • overwrite - Toujours écraser les fichiers existants
    • verbosity - Définir le niveau de verbosité
