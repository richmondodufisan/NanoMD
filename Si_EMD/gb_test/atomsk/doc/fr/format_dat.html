Format : DAT

Nom : colonnes de données

Extension : dat

Spécification : pas de standard

Programmes de visualisation : aucun

Restrictions

Un fichier au format DAT est un fichier texte contenant des colonnes de données. Lorsqu'il écrit un fichier DAT, Atomsk écrit sur chaque ligne le numéro atomique suivi des coordonnées x, y, z de chaque atome. Si des propriétés auxiliaires existent, elles sont écrites à la suite. Notez que les vecteurs de boîte et les commentaires ne sont pas écrits dans un fichier DAT.

Lorsqu'il lit un fichier DAT, Atomsk suppose qu'il respecte le format CSV, sauf que les valeurs sont séparées par des espaces au lieu de virgules. Lisez la page sur le format CSV pour plus d'information. Puisque les vecteurs de boîte ne peuvent pas être lus depuis un fichier DAT, Atomsk construira automatiquement une boîte parallélépipédique englobant tous les atomes.

Exemples

  • atomsk mon_systeme.cfg dat

    Ceci va lire le fichier mon_systeme.cfg, et le convertira en mon_systeme.dat.
