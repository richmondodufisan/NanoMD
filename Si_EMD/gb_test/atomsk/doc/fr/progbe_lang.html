Comportement du programme : langue

Syntax

-language <langue>

-lang <langue>

Description

Cette option change la langue des messages affichés par Atomsk.

Les langues disponibles sont :

  • de pour l'allemand.
  • en pour l'anglais.
  • fr pour le français.

Pour rendre ce changement permanent, vous pouvez éditer un fichier de configuration.

Défaut

Par défaut, sur les systèmes GNU/Linux la variable d'environnement $LANG est lue, et si elle correspond à un langage connu par Atomsk alors ce langage est utilisé. Sur les systèmes Microsoft® Windows® l'anglais est utilisé par défaut. Ce défaut est écrasé par le fichier de configuration si l'utilisateur en possède un. Enfin, l'option en ligne de commandes -lang écrase toutes les définitions précédentes.

Exemples

  • atomsk initial.gin -lang fr cfg

    Ceci lira le fichier initial.gin et le convertira en CFG. Tous les messages seront affichés en français.
