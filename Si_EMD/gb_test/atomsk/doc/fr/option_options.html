Option : options

Syntaxe

-options <fichier>

Description

Cette option permet de lire une liste d'options depuis un fichier.

Le <fichier> doit être un fichier texte contenant des mots-clés suivis des valeurs appropriées. Chaque mot-clé doit être une option valide, et doit être suivi de ses paramètres, dans le même ordre que s'ils étaient saisis en ligne de commandes.

Les options lues depuis le <fichier> remplaceront l'appel à l'option -options.

L'option -options elle-même ne peut pas apparaître dans le <fichier>, le cas échéant elle sera ignorée. Autrement dit cette option ne peut pas être utilisée de façon récursive. Cependant l'option -options peut être appelée plusieurs fois dans une même ligne de commandes.

Default

Par défaut aucune option n'est lue depuis un fichier.

Examples

  • mes_options.txt

    # Ceci est un commentaire
    # Ré-orienter la cellule
    -orient [110] [1-10] [001] [100] [010] [001]
    # Remplacer les types d'atomes par des espèces chimiques
    -sub 1 Si
    -sub 2 C

    atomsk SiC_relaxed.lmc -options mes_options.txt xsf

    Ceci va lire le fichier de sortie LAMMPS SiC_relaxed.lmc, appliquera les options décrites dans le fichier mes_options.txt, et écrira le résultat final dans SiC_relaxed.xsf.

  • atomsk initial.cfg -duplicate 3 3 2 -options mes_options.txt -alignx -wrap final.xsf

    Ceci va lire le fichier initial.cfg, dupliquera le système 3x3x2, puis appliquera les options décrites dans le fichier mes_options.txt, puis appliquera les options -alignx et -wrap, et enfin écrira le résultat final dans final.xsf.
