Exemples

Convertir un fichier vers un autre fichier

  • atomsk initial.cfg final.xsf

    Ceci convertira le fichier initial.cfg (au format CFG) vers le fichier final.xsf (au format XSF).

Convertir un fichier vers un autre format

  • atomsk fichier.cfg xsf

    Ceci convertira fichier.cfg (format CFG) vers fichier.xsf (format XSF).

Convertir un fichier vers plusieurs formats

  • atomsk fichier.cfg xsf lmp

    Ceci convertira fichier.cfg (format CFG) vers fichier.xsf (format XSF) et fichier.lmp (LAMMPS data).

Générer une cellule élémentaire d'aluminium cfc avec une orientation cristalline donnée

  • atomsk --create fcc 4.02 Al orient [0-11] [100] [011] cfg

    Cet exemple utilise le mode --create pour générer une cellule élémentaire d'aluminium cubique à faces centrées (cfc) avec l'orientation cristallographique X=[011], Y=[100], Z=[011].

Utiliser les options

Veuillez vous référer à la section sur les options. Plusieurs options peuvent être utiliées consécutivement dans la même ligne de commande, comme dans l'exemple ci-dessous.

  • atomsk unitcell.xyz -duplicate 20 10 1 -deform x 2 0.33 final.xsf lmp

    Dans cet exemple, Atomsk va lire le fichier unitcell.xyz (format XYZ), dupliquer le système pour former une supercellule de dimensions 20x10x1, puis le déformer de 2% suivant X (en utilisant un ratio de Poisson égal à 0.33), et enfin écrire le système final dans les fichiers final.xsf (au format XSF) et final.lmp (LAMMPS data).

Autres exemples

Le dossier "examples" fourni avec le programme contient des scripts bash commentés qui utilisent Atomsk pour construire différents systèmes. Les commentaires sont censés être ignorés par l'ordinateur, pas par vous : lisez-les pour comprendre ce que font les scripts. Bien sûr vous pouvez modifier ces scripts pour tester le programme.
