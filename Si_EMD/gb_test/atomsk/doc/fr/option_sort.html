Option : sort

Syntaxe

-sort <propriété> <ordre>

-sort random

Description

Cette option permet de trier les atomes en fonction d'une de leurs propriétés.

Les paramètres associés à cette option sont :

  • propriété : la propriété suivant laquelle les atomes seront triés, peut être 'species' ou 's' pour les trier par espèce chimique, 'x', 'y', 'z' pour les trier suivant une coordonnée, ou bien n'importe quelle propriété auxiliaire (telle que lue depuis le fichier d'entrée ou définie avec l'option -properties).
  • ordre : peut être 'up' pour trier les atomes par propriété croissante, 'down' pour les trier par ordre décroissant, ou 'pack' pour simplement rassembler les atomes ayant la même valeur de la propriété donnée (mais sans les trier). Si l'ordre est 'reverse', alors l'ordre des atomes est inversé.

Si cette option est appelée avec "-sort random", alors la liste des atomes est mélangée dans un ordre aléatoire.

Cette option ne transforme pas le système ni ne modifie les positions atomiques. Elle modifie les indices des atomes, et donc l'ordre dans lequel les atomes apparaissent dans le fichier de sortie. Par exemple, après l'application de cette option, l'atome #i de symbole chimique Xx à la position (x,y,z) et avec des valeurs de propriétés auxiliaires A1, A2, etc., sera toujours de la même espèce chimique, il sera à la même position, et aura les mêmes valeurs de ses propriétés auxiliaires, mais il aura un indice #j différent.

Défaut

Par défaut les atomes ne sont pas triés.

Exemples

  • atomsk initial.cfg -sort species up final.xyz

    Ceci triera les atomes par numéro atomique croissant.

  • atomsk initial.cfg -sort z down final.xyz

    Ceci triera les atomes par coordonnée Z décroissante.

  • atomsk initial.cfg -sort s pack POSCAR

    Ceci rassemblera les atomes ayant la même espèce chimique, de sorte qu'ils soient contigus dans le fichier final.

  • atomsk initial.lmc -sort vx up final.xyz exyz

    Ceci lira le fichier initial.lmc et triera les atomes par valeur croissante de leur vitesse suivant X. Le résultat final sera écrit dans le fichier final.xyz au format XYZ étendu.

  • atomsk initial.lmc -sort random final.cfg

    Ceci va mélanger aléatoirement les atomes, c'est-à-dire que tous leurs indices seront échangés de telle sorte à apparaître dans un ordre aléatoire.
