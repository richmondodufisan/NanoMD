Option : alignx

Syntaxe

-alignx

Description

Cette option aligne le premier vecteur de boîte avec l'axe cartésien X, et le second vecteur de boîte avec le plan (X,Y). Autrement dit elle fait pivoter le système de sorte que les vecteurs de boîte forment une matrice triangle H' :

| H'(1,1) 0.00000 0.00000 |
| H'(2,1) H'(2,2) 0.00000 |
| H'(3,1) H'(3,2) H'(3,3) |

Bien entendu, les atomes sont aussi pivotés en conséquence.

Cette option affectera certaines propriétés du système (par ex. le tenseur élastique) si elles avaient été précédemment définies par l'option -properties.

Notez que cette option ne modifie pas le système, elle ne fait que le tourner.

Défaut

Par défaut les vecteurs de boîte ne sont pas ré-alignés.

Exemples

  • atomsk system.gin -alignx xsf

    Ceci lira le fichier system.gin, ré-alignera ses vecteurs de boîte, et écrira le résultat final dans system.xsf.
