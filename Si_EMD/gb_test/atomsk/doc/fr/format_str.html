Format : STR

Nom : fichier de structure PDFFIT

Extension : str ou stru

Spécification : guide d'utilisateur de PDFFIT

Programmes de visualisation : N/A

Restrictions

PDFFIT est un programme permettant de déterminer une structure cristalline à partir de clichés de diffraction de rayons X. Après avoir effectué un raffinement, la structure finale du cristal est enregistrée dans un fichier texte au format STR.

Atomsk peut lire les fichiers de structure aussi bien dans le format utilisé par DISCUS, que dans celui utilisé par PDFFIT. Lors de la lecture de ces fichiers, les occupations partielles et les facteurs de vibration anisotropes sont enregistrés, de sorte à pouvoir être transférés dans les fichiers de sortie si leur format le permet. Si un groupe d'espace est défini (après le mot-clé "spcgr"), alors les opérations de symétrie correspondantes sont appliquées immédiatement.

Atomsk peut aussi écrire des fichiers au format STR, en utilisant toujours le format de PDFFIT. Le groupe d'espace est toujours P1, et toutes les positions atomiques sont écrites explicitement dans le fichier. S'ils sont définis, les occupations partielles et facteurs anisotropes sont aussi écrits dans le fichier ; toute valeur manquante est automatiquement remplacée par un zéro. De plus, les déviations standards pour tous les paramètres sont toujours égales à zéro.

Exemples

  • atomsk Ni.str vesta

    Avec cette commande, Atomsk va lire le fichier Ni.str, et le convertir en Ni.vesta pour visualisation avec VESTA. Les occupations partielles seront enregistrées et écrites dans le fichier de sortie, et pourront donc être visualisées dans VESTA.
