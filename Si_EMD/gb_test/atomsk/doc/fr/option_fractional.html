Option : fractional

Syntaxe

-fractional

Description

Cette option convertit toutes les positions des atomes en coordonnées réduites (aussi appelées coordonnées fractionnaires), c-à-d. en coordonnées normalisées aux vecteurs de boîte.

De fait les coordonnées seront écrites en coordonnées fractionnaires dans le fichier de sortie, quel que soit le format. Par exemple, par défaut Atomsk écrit les coordonnées en base cartésienne dans un fichier au format XYZ, mais avec cette option elles seront écrites en coordonnées réduites. Pour certains formats (comme le CFG) les positions sont toujours écrites en coordonnées réduites.

Cette option peut affecter certains modes. Par exemple si elle est utilisée avec le mode difference alors les déplacements calculés seront en coordonnées réduites. Certaines options peuvent aussi agir différemment si elles sont appelées après l'option -fractional.

Cette option ne fera rien si les positions des atomes sont déjà en coordonnées réduites (par ex. si cette option a déjà été appelée avant).

Défaut

Par défaut, Atomsk utilise des coordonnées cartésiennes en interne, et n'écrit des coordonnées réduites que dans les formats de fichier qui le requièrent explicitement (comme le format CFG d'Atomeye).

Exemples

  • atomsk sto.xsf -fractional sto.xyz

    Ceci lira le fichier sto.xsf et écrira des coordonnées réduites vers sto.xyz.

  • atomsk sto.xsf -frac sto.gin

    Ceci lira le fichier sto.xsf et écrira des coordonnées réduites vers sto.gin.
