Option : spacegroup

Syntaxe

-spacegroup <groupe>

Description

Cette option applique les opérations de symétrie du groupe d'espace donné au système.

Le <groupe> peut être soit le numéro du groupe d'espace, ou bien son symbole de Hermann-Mauguin (sans espace). Après l'application des opérations de symétrie, les atomes situés trop proches d'autres atomes sont supprimés.

Notez que cette option change le nombre d'atomes dans le système (mais ne change pas les vecteurs de boîte).

Défaut

Les fichiers au format CIF, au format GULP, au format JEMS, ou au format VESTA, peuvent contenir un numéro (ou symbole) d'un groupe d'espace. Lorsque c'est le cas, Atomsk applique les symétries de ce groupe d'espace immédiatement à la lecture de ces fichiers. Sinon, par défaut aucune opération de symétrie n'est appliquée.

Exemples

  • atomsk initial.cfg -spacegroup Pnma final.xyz

    Avec cette commande, Atomsk lit le fichier "initial.cfg", applique les opérations de symétrie du groupe d'espace Pnma, et écrit le résultat final dans le fichier "final.xyz".

  • atomsk initial.cfg -spacegroup 42 final.xyz

    Ceci appliquera les opérations de symétrie du groupe d'espace numéro 42 (Fmm2).
