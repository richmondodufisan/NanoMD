Option : remove-shells

Syntaxe

-remove-shells <espèce>

-rmshells <espèce>

Description

Cette option supprime les coquilles (dans le cadre d'un potentiel ionique cœur-coquille) des ions de l'espèce donnée. Si <espèce> vaut "all" ou "any" alors toutes les coquilles sont retirées du système.

Si une sélection a été définie (grâce à l'option -select) alors les coquilles ne seront supprimées que sur les ions sélectionnés.

Défaut

Par défaut aucune coquille n'est supprimée.

Exemples

  • atomsk initial.gin -remove-shells Ti final.gin

    Ceci va lire le fichier GULP initial.gin, supprimera les coquilles des ions titane, et écrira le résultat dans le fichier GULP final.gin.

  • atomsk REVCON -rmshells all final.gin

    Ceci supprimera toutes les coquilles du système.
