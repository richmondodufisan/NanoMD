Mode : aide

Syntaxe

atomsk --help [<sujet>]

Description

Ce mode affiche un bref résumé de l'utilisation du programme.

Un sujet particulier peut être donné pour afficher l'aide le concernant : "modes", "options", "formats".

Notez que si un mode ou une option est invoquée de façon incorrecte, son usage sera affiché.

Aucun fichier n'est traité ni converti dans ce mode (et toute option ou format de sortie sont donc ignorés).

Exemples

  • atomsk --help

    Ceci affichera l'aide générale.

  • atomsk --help options

    Ceci affichera la partie de l'aide correspondant aux options.

  • atomsk --merge

    Ceci affichera la partie de l'aide correspondant au mode merge.

  • atomsk -shift

    Ceci affichera la partie de l'aide correspondant à l'option -shift.
