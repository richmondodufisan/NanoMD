Format : OUTCAR

Nom : fichier de sortie de Vienna Ab initio Simulation Package (VASP)

Extension : aucune (le fichier est nommé OUTCAR par VASP)

Spécification : documentation de VASP

Programmes de visualisation : aucun

Restrictions

Atomsk peut lire les fichiers OUTCAR (tels que spécifiés dans VASP versions 4.x et 5.x) seulement dans le mode "--one-in-all". Ce mode lit toutes les configurations contenues dans le fichier OUTCAR, et écrit chacune d'elles dans un fichier séparé. Les forces agissant sur les atomes seront aussi lues depuis le fichier OUTCAR, et seront écrites dans les fichiers de sortie si leur format le permet.

Notez que le fichier OUTCAR ne contient plusieurs configurations que si VASP a effectué des pas de relaxation atomique ou de dynamique moléculaire (IBRION≥0). Sinon, le fichier OUTCAR ne contient que des informations sur la structure électronique, que Atomsk ne peut pas lire.

Atomsk ne peut pas écrire de fichier au format OUTCAR.

Exemples

  • atomsk --one-in-all OUTCAR cfg

    Avec cette commande, Atomsk va lire le fichier OUTCAR, et écrira chaque configuration dans un fichier séparé au format CFG, qui s'appeleront OUTCAR_0.cfg, OUTCAR_1.cfg, OUTCAR_2.cfg, et ainsi de suite. Les forces agissant sur les atomes seront aussi importées, et écrites dans les fichiers CFG.

  • atomsk --one-in-all OUTCAR vesta

    Ceci va lire le fichier OUTCAR et écrire les configurations au format VESTA pour visualisation.
