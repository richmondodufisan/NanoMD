Modes

Les modes déterminent la manière dont Atomsk traite les fichiers, par ex. convertir un fichier vers d'autres formats, réunir plusieurs fichiers en un seul... Un seul mode peut être utilisé à la fois.

Si le programme est invoqué sans aucun argument en ligne de commande (i.e. si vous tapez juste "atomsk") alors le programme entrera en mode interactif.

Si des arguments sont donnés dans la ligne de commande mais qu'aucun mode spécial n'est spécifié, alors le programme s'exécutera en mode normal.

Les autres modes doivent être appelés explicitement avec leur nom, toujours précédés de deux tirets, de la forme --<mode>. Veuillez vous référer à la documentation de chaque mode ci-dessous.

  • Average - Moyenner les positions atomiques de plusieurs configurations
  • Copy-Properties - Copier les propriétés d'un systèem à un autre
  • Create - Créer un système à partir de rien
  • ddplot - Créer un fichier pour visualisation avec le programme ddplot
  • Gather - Rassembler plusieurs fichiers vers un seul contenant toutes les configurations
  • Liste - Convertir tous les fichiers d'une liste
  • Help - Afficher l'aide
  • Interactive - Manipuler des systèmes interactivement ou avec des scripts
  • Interpolate - Interpoler les positions des atomes entre deux configurations
  • Match-ID - Trouver des atomes équivalents entre deux systèmes
  • Merge - Joindre deux ou plusieurs systèmes en un seul
  • Normal - Convertir un fichier vers un ou plusieurs formats
  • Polycrystal - Créer un polycristal avec la méthode de Voronoi
  • Unfold - Décomposer un fichier contenant plusieurs systèmes vers plusieurs fichiers
  • Unwrap - Déballer les coordonnées atomiques

Les modes ci-dessous réalisent des opérations particulières sur un ou plusieurs systèmes.

  • Densité - Calculer la densité d'une propriété
  • Difference - Calculer la difference entre les positions atomiques de deux configurations
  • Electric dipole moments - Calculer des moments dipolaires électriques
  • Electronic polarization - Calculer la polarisation ionique dans un modèle ionique cœur-coquille
  • Symétrie locale - Calculer un paramètre de symétrie locale pour chaque atome
  • Nye tensor - Calculer le tenseur de Nye
  • RDF - Calculer la fonction de distribution radiale
